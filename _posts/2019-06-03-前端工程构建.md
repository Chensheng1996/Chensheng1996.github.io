### 构建Node工程
#### 1.安装Express框架
```node
npm install express -g  
npm install express-generator -g
```
#### 2.新建一个项目
```node
express -t ejs ProjectName // 使用ejs模板
```
#### 3.安装项目并启动
```node
npm install // 安装，生成node_modules
npm start // 端口号默认3000
```
